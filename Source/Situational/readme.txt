---lazyMapperDoubleSpeed.py---
Doubles the BPM for slow songs. Example: 95 becomes 190 BPM

---lazyMapperForceStandard.py---
Forces CS4 AR9 OD8 HP5 no matter the selected map.

---lazyMapperHalfSpeed.py---
Halves the BPM for fast songs. Example: 360 becomes 180 BPM

---lazyMapperSmallTitle.py---
Only in case Osu! crashes when trying to generate. It might be because the file name is too big.
Reduces the file name, and does not affect the difficulty name.
However, you can only create this once. Creating another one will replace the previous one.
